Filter an array based on user-\/defined condition. 

\begin{DoxyNote}{Note}
You will need to use ctypes in the filter function to convert values so the underlying C code knows what to do. 

Currently this filter doesn\textquotesingle{}t support different filters on different columns simultaneously, but that\textquotesingle{}s planned soon. 
\end{DoxyNote}

\begin{DoxyParams}{Parameters}
{\em filter\+\_\+func} & A user-\/defined python function that takes one parameter and returns a boolean. You will need to use ctypes to convert this parameter into your array type. See example below. \\
\hline
{\em secondary\+\_\+indices} & These are the indices to restrict the filter to and are analogous to columns. E.\+g if you pass \mbox{[}1\mbox{]} it will only check the filter against column 1. If you pass an empty list \mbox{[}\mbox{]}, it will check all columns. \\
\hline
{\em filter\+\_\+type} & A string specifying \textquotesingle{}ANY\textquotesingle{} or \textquotesingle{}ALL\textquotesingle{}. This only applies to arrays 2D or above and if you are applying the filter to multiple columns. If \textquotesingle{}ANY\textquotesingle{} is used, then the filter must pass (be true) for AT LEAST one of the columns; then that row will be returned. If \textquotesingle{}ALL\textquotesingle{} is used, then ALL columns must satisfy the filter in order for that row to be returned.\\
\hline
\end{DoxyParams}
